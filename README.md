# 🤖 BISINDO AI Translator

Real-Time Indonesian Sign Language (BISINDO) Translator using **MediaPipe Hand Landmarker** and **Deep Learning Neural Network**.

## ✨ Features

- 🖐 Real-Time BISINDO Recognition
- 🧠 Deep Learning Neural Network
- 🎥 MediaPipe Hand Landmarker
- 🔒 Gesture Lock
- 📝 Automatic Sentence Builder
- 🔊 Auto Text-to-Speech
- 📊 Confidence Score
- ⚡ Real-Time Prediction

---

## 📊 Model

| Item | Value |
|------|------|
| Classes | 43 |
| Dataset | 28,685 Images |
| Input | 126 Hand Landmark Features |
| Accuracy | **97.96%** |

